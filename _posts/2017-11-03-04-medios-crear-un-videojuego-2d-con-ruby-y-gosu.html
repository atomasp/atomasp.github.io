Comenzaremos por presentar los medios que se van a utilizar en el juego.

Como hemos definido en las especificaciones, la pantalla tendrá un tamaño de 800x600. A esta pantalla le vamos a añadir un fondo de pantalla para simular la pared de la mazmorra. Para ello, hemos utilizado un diseño "flat" para crear nuestro propio fondo:


Este fondo de pantalla lo vamos a nombrar "bg.png" y lo vamos a almacenar en una carpeta llamada "media". Por tanto, para mostrar el fondo de pantalla en nuestro juego lo haremos de la siguiente manera: 

require "gosu"

class Game < Gosu::Window
  def initialize
    super 800, 600
    self.caption = "Juego de las campanas"
    @background_image = Gosu::Image.new("media/bg.png", :tileable => true)
  end

  def update
  end

  def draw
    @background_image.draw(0, 0, 0)
  end
end

window = Game.new
window.show

Es decir, generamos el fondo de pantalla en el constructor (método initialize()) y luego lo dibujamos en el método draw().
