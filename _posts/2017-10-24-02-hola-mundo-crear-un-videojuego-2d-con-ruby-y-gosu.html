- 01 Prerrequisitos: Los prerrequisitos iniciales para poder comenzar a desarrollar nuestro juego con Ruby y Gosu.

Tras la instalación de la gema de Gosu como hemos visto en el post anterior, vamos a proceder a crear nuestro primer juego, el juego "Hola mundo!".

Para ello, crearemos con nuestro editor de código favorito un archivo con extensión ".rb" que contendrá el siguiente código: 

require 'gosu'

class Juego < Gosu::Window
  def initialize
    super 640, 480
    self.caption = "Hola mundo!"
  end

  def update
  end

  def draw
  end
end

window = Juego.new
window.show

Esta es la estructura básica de un programa en Gosu, primero añadimos la librería de Gosu y posteriormente creamos una clase ("Juego" en nuestro ejemplo) que hereda de "Gosu::Window". Esta clase contiene, a parte del constructor (método "initialize()"), otros dos métodos "update()" y "show()" que sobrescriben los métodos definidos por "Gosu::Window".

"update()" se ejecuta 60 veces por segundo por defecto y tiene que contener la lógica del juego. Por otro lado, "draw()" se ejecuta a continuación de "update()" y debe contener el código para redibujar la pantalla, pero no debe actualizar ningún estado del juego.

Para poder ejecutar el juego tenemos que crear un nuevo objeto "Juego" y llamar al método "show()". Esto mostrará la ventana del juego hasta que se cierre o se llame al método "close()".
