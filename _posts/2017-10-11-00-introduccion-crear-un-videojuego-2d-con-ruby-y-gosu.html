--- layout: post title: 00 Introducción - Crear un videojuego 2D con Ruby y Gosu date: '2017-10-11T12:08:00.000+02:00' author: AITOR TOMAS PIN tags: modified_time: '2017-12-28T12:11:07.617+01:00' blogger_id: tag:blogger.com,1999:blog-611028137431305192.post-295224770258077382 blogger_orig_url: http://compupiensaumh.blogspot.com/2017/12/00-introduccion-crear-un-videojuego-2d-con-ruby-y-gosu.html --- En los últimos años es tendencia el uso de MIT Scratch para introducir a los alumnos conceptos de lógica de programación. Scratch es un recurso fantástico, muy adecuado para alumnos sobre todo de ESO, pero si queremos ir un paso más allá y dar continuidad a los conocimientos adquiridos por el uso de Scratch, puede resultar interesante el uso de un lenguaje de programación como puede ser Ruby. Por su sencillez, Ruby es una buena alternativa para continuar con el proceso de aprendizaje en materia de programación.

No es objetivo de este post el introducir Ruby. Existen infinitos recursos en la red donde aprender acerca de Ruby. Nos centraremos sin embargo en el uso de la gema Gosu (en Github) para poder crear videojuegos en 2D. Gosu es una librería para Ruby y C++ para el desarrollo de videojuegos en 2D. Ésta nos va a permitir el desarrollo de un simple videojuego el cual se podría realizar de manera colaborativa con los alumnos.

Se propone para la realización del ejercicio el uso de herramientas colaborativas como Git y Github.

¿Qué van a aprender los alumnos?
- Trabajar en equipo.
- El uso de herramientas colaborativas reales como Git y Github.
- Programación orientada a objetos.
- Resolución de problemas de lógica.
- Desarrollo artístico.
