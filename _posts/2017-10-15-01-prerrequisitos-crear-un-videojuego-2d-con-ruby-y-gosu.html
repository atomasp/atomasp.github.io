Como hemos mencionado en el primer post, vamos a utilizar herramientas colaborativas como Git y Github para tanto almacenar nuestro código fuente, como para poder trabajar de manera colaborativa. Para ello es necesario crear una cuenta en Github y crear un repositorio para albergar nuestro proyecto. Se recomienda la lectura del siguiente recurso sobre cómo trabajar con Github de manera colaborativa mediante el uso de "branches" y "pull requests" (modelos de desarrollo colaborativo). Así mismo, se recomienda para poder utlizar Github de manera eficaz el aprender a utilizar Git. Para ello, el siguiente video muestra de manera muy gráfica cómo funciona Git:


Por otro lado, vamos a considerar que Ruby ya está instalado y vamos a proceder a instalar la gema Gosu. Para ello, podemos seguir los pasos descritos en el Wiki de Gosu en Github dependiendo de cuál es nuestro sistema operativo Mac OS X, Windows o Linux. En nuestro caso, vamos a utilizar Mac OS X, por tanto:

Instalamos la librería sdl2: 

brew install sdl2

Y una vez instalado sdl2, instalamos la gema de Gosu: 

gem install gosu

Para comprobar que todo funciona correctamente, podemos ejecutar: 

ruby -rgosu -e 'w = Gosu::Window.new(200, 150); w.caption = "It works!"; w.show'

Si se abre una ventana con título "It works!", entonces estamos preparados para empezar.

Dejamos a elección de cada persona el decidir que editor de código utilizar. Sin embargo, recomendamos el recurso de Github for education y en particular su Pack para Estudiantes. Este Pack permite el accesso a herramientas reales de desarrollo de manera gratuita para estudiantes.
